# Sum Program Example (Ontology Style)

This folder contains instance examples showing how the classic "read two integers, compute their sum, and print the result" program can be represented using the new Golang ontology.

These are **instances** (not classes) that the planner can compose.

## Files

- `main_function.json` — The main function
- `var_declarations.json` — Variable declarations for a and b
- `input_acquisition.json` — Reading values using fmt.Scanf
- `addition.json` — The addition operation
- `output.json` — Printing the result
- `full_program.json` — A composed view of the entire program
